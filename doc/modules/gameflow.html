gameflow.lua

These functions are called in gameflow.lua, a file loosely equivalent to winroomedit's SCRIPT.DAT. They handle a game's 'metadata'; i.e., things such as level titles, loading screen paths, and default ambient tracks.
AddLevel (level)
Add a level to the gameflow.

Parameters:

  • level Level a level object
SetIntroImagePath (path)
The path of the .jpg or .png image to show when loading the game.

Parameters:

  • path string the path to the image, relative to the TombEngine exe
SetTitleScreenImagePath (path)
The path of the .jpg or .png image to show in the background of the title screen. (not yet implemented)

Parameters:

  • path string the path to the image, relative to the TombEngine exe
SetWindowTitleKey (string)
The string ID of the title of the game (not yet implemented)

Parameters:

  • string string ID of the title as set in TombIDE.
