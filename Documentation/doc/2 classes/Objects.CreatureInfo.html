Class Objects.CreatureInfo

Represents the AI and behavior state of a creature in the game.

Functions

CreatureInfo(mov) Create creature info for the provided moveable.
GetMood() Gets the current mood of the creature.
GetTarget() Gets the current target of the creature.
GetTargetPosition() Gets the current target position of the creature.
SetTarget(mov) Sets a new target for the creature.
SetTargetPosition(position) Sets the position of the creature's target.
ClearTarget() Clears the current target of the creature.
IsAlerted() Checks if the creature is in an alerted state.
IsFriendly() Checks if the creature is friendly.
IsHurtByPlayer() Checks if the creature has been hurt by player.
IsPoisoned() Checks if the creature is poisoned.
IsAtGoal() Checks if the creature has reached its goal.

Create creature info for the provided moveable.

Parameters:

    CreatureInfo Creature info for the moveable.

IsFriendly()
Checks if the creature is friendly. Only returns true for friendly creatures like monks (TR2) or troops (TR4).

Returns:

    Creature friendly status. bool true: if the creature is friendly, false: not friendly

Checks if the creature has been hurt by player.

Checks if the creature is poisoned.
